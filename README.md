# 好漢社

好漢社は、\
男性が誇りをもって立派に生きられる社会を願う人の集まりです。

誇りを持ち、責任を引き受け、\
社会に静かに貢献する男性が増えることは、\
誰にとっても不利益にはなりません。

私たちは誰かを糾弾するために集まるのではありません。

ただ、この社会のあり方に\
違和感を覚えている人が、どれくらいいるのか。\
それを静かに可視化するための場です。

趣旨に賛同される方は、\
連絡先としてメールアドレスをご登録ください。\
性別・国籍・立場は問いません。\
登録による義務や費用はありません。

# 関心事項

好漢社は、\
個人を性別で単純化し、\
役割や評価を固定する考え方とは距離を取ります。

その観点から、\
近年のいくつかの施策や広報表現について、\
違和感を覚える人が少なくないのも事実です。

- 冤罪・司法における不均衡
- 政治や行政へのアクセス（声の届き方）の不均衡
- 大学や就職（とくに公的部門）における不均衡
- 補助金や支援事業等の不均衡
- 交通や公共空間における不均衡
- 生理痛体験
